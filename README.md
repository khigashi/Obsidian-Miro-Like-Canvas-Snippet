# Obsidian Miro Like Canvas Snippet

This snippet gives Obsidian Canvas a Miro-like look, while keeping changes lightweight to reduce incompatibilities. It was made for my own setup, but feel free to copy and tweak it.

## Installation

1. Place `obsidian-miro-like-canvas-snippet.css` in:

   - `<your-vault>/.obsidian/snippets/`

2. In Obsidian, enable it at:

   - `Settings` -> `Appearance` -> `CSS snippets`
